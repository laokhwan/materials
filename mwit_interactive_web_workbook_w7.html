Mini Project: Personal Study Planner

ภารกิจ: สร้างเว็บแอปที่รับข้อมูลแผนการอ่านหนังสือ คำนวณเวลาเฉลี่ยต่อบท แสดงคำแนะนำ และบันทึกข้อมูลล่าสุดด้วย Local Storage

HTML ควรมี

  • input type="text" สำหรับชื่อและวิชา
  • input type="number" สำหรับจำนวนบทและจำนวนชั่วโมง
  • select สำหรับระดับความยาก
  • textarea สำหรับบันทึกเพิ่มเติม
  • button สำหรับ Calculate / Save / Load / Clear
  • div id="result" สำหรับแสดงผลลัพธ์

JavaScript ควรทำได้

  • อ่านค่าจาก input ด้วย .value
  • แปลงตัวเลขด้วย Number()
  • คำนวณเวลาเฉลี่ยต่อบท
  • ใช้ if/else เพื่อสร้างคำแนะนำ
  • บันทึกและโหลดด้วย localStorage
Challenge สำหรับผู้เรียนเร็ว: เพิ่มประวัติการบันทึกหลายรายการ, เพิ่มสีตามความเร่งด่วน, เพิ่มปุ่ม Export ข้อมูลเป็นข้อความ หรือเพิ่มระบบนับจำนวนครั้งที่เปิดเว็บนี้

Checkoff และ Reflection

  • อธิบายได้ว่า HTML, CSS และ JavaScript มีหน้าที่ต่างกันอย่างไร
  • สร้าง input อย่างน้อย 3 ชนิดได้
  • ตกแต่ง input และ button ด้วย CSS ได้
  • อ่านค่าจาก input ด้วย JavaScript ได้
  • แปลงข้อมูลเป็นตัวเลขด้วย Number() ได้
  • ใช้ if/else เพื่อสร้างคำแนะนำได้
  • ใช้ localStorage เพื่อ Save / Load / Clear ข้อมูลได้

Reflection

  1. HTML ส่วนใดในเว็บของคุณเป็น input?
  2. CSS ส่วนใดที่ช่วยให้เว็บอ่านง่ายหรือใช้งานง่ายขึ้น?
  3. JavaScript อ่านค่าจาก input ใดบ้าง?
  4. ข้อมูลใดต้องใช้ Number() ก่อนคำนวณ?
  5. Local Storage เหมาะกับข้อมูลประเภทใด และไม่เหมาะกับข้อมูลประเภทใด?
