Mini Project: Personal Study Planner

ภารกิจ: สร้างเว็บแอปที่รับข้อมูลแผนการอ่านหนังสือ คำนวณเวลาเฉลี่ยต่อบท แสดงคำแนะนำ และบันทึกข้อมูลล่าสุดไว้ใน Local Storage โดยแยกไฟล์เป็น index.html, style.css, script.js

HTML ควรมี

  • input type="text" สำหรับชื่อและวิชา
  • input type="number" สำหรับจำนวนบทและจำนวนชั่วโมง
  • select สำหรับระดับความยาก
  • radio สำหรับเป้าหมายการอ่าน
  • checkbox สำหรับวิธีอ่าน
  • textarea สำหรับบันทึกเพิ่มเติม
  • button สำหรับ Calculate / Save / Load / Clear

CSS ควรมี

  • กำหนดฟอนต์และพื้นหลังของหน้าเว็บ
  • ตกแต่งกล่องฟอร์มด้วย padding, border, border-radius
  • ตกแต่ง input และ button ให้ใช้งานง่าย
  • ทำกล่องแสดงผลให้เด่นและอ่านง่าย

ตัวอย่างแนวทางไฟล์

study-planner
├── index.html   // โครงสร้างฟอร์ม
├── style.css    // สี รูปแบบ ระยะห่าง
└── script.js    // อ่านค่า คำนวณ แสดงผล บันทึกข้อมูล

Checkoff และ Reflection

  • ตกแต่ง input และ button ด้วย CSS ได้
  • อ่านค่าจาก input ด้วย JavaScript ได้
  • แปลงข้อมูลเป็นตัวเลขด้วย Number() ได้
  • ใช้ if / else เพื่อสร้างคำแนะนำได้
  • ใช้ localStorage เพื่อ Save / Load / Clear ข้อมูลได้
