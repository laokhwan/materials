Lab 5: บันทึกข้อมูลด้วย Local Storage

localStorage ใช้บันทึกข้อมูลไว้ใน Browser ข้อมูลจะยังอยู่แม้ Refresh หน้าเว็บ แต่ข้อมูลนี้อยู่เฉพาะเครื่องและ Browser นั้น ๆ

Live Preview

Modify: เปลี่ยน key จาก "note" เป็น "myPlan" ทั้งใน Save และ Load แล้วทดลองอีกครั้ง

คำสั่งสำคัญ

parent.localStorage.setItem("note", text) บันทึกข้อมูลผ่านหน้าต่างหลัก

parent.localStorage.getItem("note") โหลดข้อมูลจากหน้าต่างหลัก

parent.localStorage.removeItem("note") ลบข้อมูล

💡 ตัวอย่างไอเดียชิ้นงานที่สามารถเลือกสร้างได้:

2. Financial Helper (เครื่องคำนวณเงินออม/รายจ่าย)

แอปคำนวณเป้าหมายเงินออมประจำวัน/เดือน รับค่ารายได้และเป้าหมาย สรุปผลว่าต้องออมวันละเท่าไหร่ พร้อมเปลี่ยนสีกล่องเตือนตามความยากง่ายของเป้าหมาย

ข้อกำหนดด้านระบบ (Technical Requirements)

  • HTML: มี input ชนิดต่าง ๆ เช่น text, number, select หรืออื่นๆ และมีปุ่มอย่างน้อย 2 ปุ่ม (คำนวณ / บันทึก)
  • CSS: จัดรูปแบบ Layout ให้อ่านง่าย มีการใช้ padding, margin, border-radius และออกแบบสถานะผลลัพธ์ให้น่าสนใจ
  • JavaScript: ดึงข้อมูลจาก UI มาคำนวณทางคณิตศาสตร์อย่างถูกต้อง มีการใช้ if/else เพื่อจำแนกประเภทคำแนะนำ และใช้ localStorage บันทึกและโหลดข้อมูลกลับมาได้

คำแนะนำในการเริ่มทำโครงงาน

สร้างโฟลเดอร์โครงงานบนโปรแกรม VS Code แยกไฟล์ออกเป็น 3 ส่วนหลักเพื่อให้ง่ายต่อการจัดการ:

my-web-app/
├── index.html   <-- โครงสร้างฟอร์มรับค่า
├── style.css    <-- ชุดตกแต่งสีและหน้าตา
└── app.js       <-- ระบบคำนวณและจดจำข้อมูล

Checkoff และ Reflection

Checkoff สำหรับตรวจสอบตนเอง

  • อธิบายได้ว่า HTML, CSS และ JavaScript มีหน้าที่ต่างกันอย่างไร
  • สร้าง input ด้วย HTML ได้อย่างน้อย 3 ชนิด
  • ตกแต่ง input และ button ด้วย CSS ได้
  • อ่านค่าจาก input ด้วย JavaScript ได้
  • แปลงข้อมูลเป็นตัวเลขด้วย Number() หรือ parseFloat() ได้
  • ใช้ if / else เพื่อสร้างคำแนะนำได้
  • ใช้ localStorage เพื่อ Save / Load / Clear ข้อมูลได้

Reflection (ชวนคิดหลังทำกิจกรรม)

  1. ในโครงงานที่นักเรียนสร้างขึ้น HTML ส่วนใดทำหน้าที่เป็น Input และส่วนใดทำหน้าที่เป็น Output?
  2. การเลือกใช้สีหรือการปรับระยะห่าง (CSS) มีส่วนช่วยให้ผู้ใช้งานแอปพลิเคชันของเราเข้าใจผลลัพธ์ได้ง่ายขึ้นอย่างไร?
  3. เหตุใดเราจึงต้องใช้ฟังก์ชันจำพวก Number() หรือ parseFloat() ก่อนนำค่าจากหน้าเว็บไปคำนวณ? จะเกิดอะไรขึ้นถ้าลืมใช้?
  4. Local Storage เหมาะสมกับการเก็บข้อมูลประเภทใดในชีวิตจริง และข้อมูลประเภทใดที่ไม่ควรจัดเก็บลงบน Local Storage เป็นอันขาด? เพราะเหตุใด?
