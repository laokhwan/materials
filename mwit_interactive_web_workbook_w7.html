  • button สำหรับ Calculate / Save / Load / Clear
  • div สำหรับแสดงผลลัพธ์

CSS ควรมี

  • ตกแต่งพื้นหลังและกล่องฟอร์ม
  • ตกแต่ง input, select, button
  • ทำกล่องแสดงผลให้อ่านง่าย
