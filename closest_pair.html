วิธีแก้: ตรวจแถบรอบเส้นแบ่ง (Strip)

กำหนดแถบกว้าง 2δ รอบเส้นแบ่ง (δ = ระยะที่ใกล้สุดจากทั้งสองฝั่ง) ตรวจเฉพาะจุดที่อยู่ในแถบนี้เท่านั้น

✓

ผลลัพธ์

พบว่า E–F ใกล้กันกว่าคู่ที่ดีที่สุดในแต่ละฝั่ง จึงเป็นคำตอบสุดท้าย

ขั้นที่ 1: δ (delta) คืออะไร และหาได้อย่างไร?
δ (delta) = ระยะทางระหว่างคู่จุด 2 จุด ที่อยู่ใกล้กันที่สุดในฝั่งนั้น   (ไม่ใช่ระยะจากจุดไปยังเส้นแบ่ง)

ฝั่งซ้าย

คู่ใกล้สุดในฝั่งซ้าย: B–C  →  δ_left = 1.8

ฝั่งขวา

คู่ใกล้สุดในฝั่งขวา: E–F  →  δ_right = 1.5
ดังนั้น   δ = min(δ_left, δ_right) = min(1.8, 1.5) = 1.5    ← ใช้ค่านี้กำหนดความกว้างของแถบ Strip
ขั้นที่ 2–4: สร้างแถบ → เรียงจุด → ตรวจคู่ข้ามฝั่ง
2

สร้างแถบ Strip

ลากเส้นขนานกับเส้นแบ่ง ทั้งซ้ายและขวา ห่างออกมาข้างละ δ = 1.5 ซม.
เก็บเฉพาะจุดที่อยู่ในแถบกว้าง 2δ นี้เข้าไว้

>> แถบกว้าง 2δ = 3.0 ซม. รอบเส้นแบ่ง
จุดที่อยู่นอกแถบไม่มีทางใกล้กันน้อยกว่า δ ได้ จึงข้ามได้ทันที
3

เรียงจุดในแถบจากบนลงล่าง

นำจุดในแถบมาเรียงตามตำแหน่งแนวตั้ง (จากบนลงล่าง)
เพื่อตรวจเฉพาะจุดที่อยู่ใกล้กันในแนวตั้งด้วย

>> เช่น จุดที่อยู่ในแถบ: [P(y=1.2), Q(y=1.8), R(y=3.5)]
P–Q ห่างกันในแนวตั้งแค่ 0.6
P–R ห่างกัน 2.3 > δ ไม่ต้องตรวจ
4

ตรวจแต่ละคู่ในแถบ

ตรวจแต่ละจุดในแถบกับจุดถัดไปไม่เกิน 7 จุด
ถ้าพบคู่ที่ใกล้กว่า δ ให้อัปเดต δ ใหม่ทันที

>> วัด P–Q ได้ 1.2 < δ=1.5 → อัปเดต δ=1.2
คู่นี้คือ "คู่ข้ามฝั่ง" ที่ D&C ปกติจะมองข้ามไป
