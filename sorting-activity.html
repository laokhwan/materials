📋 Pseudocode — Bubble Sort
procedure bubble_sort(arr):
n ← length of arr
repeat
swapped ← false
for i from 0 to n−2:
if arr[i] > arr[i+1]:
swap arr[i] and arr[i+1]
swapped ← true
until swapped = false
สังเกต: ถ้า pass ใดไม่มีการ swap เลย (swapped = false) แปลว่าเรียงเสร็จแล้ว ออกได้เลยโดยไม่ต้องทำต่อ
